## **Epoch**

```
所有训练样本在神经网络中都进行了一次正向传播和一次反向传播。也就是1个epoch等于使用训练集中的全部样本训练一次。
```



## Batch

```
将整个训练样本分成若干个Batch。
```



## Iteration

```
1个iteration等于使用batch size个样本训练一次。一个迭代=一个正向通过+一个反向通过。训练一个Batch就是一次Iteration。
```



## Batch_Size

```
每批样本的大小，即每次训练在训练集中取 batchsize 个样本进行训练。
```
